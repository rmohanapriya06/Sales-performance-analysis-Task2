SALES PERFORMANCE ANALYSIS - TASK 2

This project analyzes sales performance using the Superstore dataset as part of Alfido Tech’s Data Analytics internship tasks. The goal is to train interns in data cleaning, exploratory analysis, visualization, and actionable business reporting using real-world datasets.

KEY FEATURES:

Data cleaning and preprocessing (handling missing values, duplicates, date formatting)
Sales KPIs calculation (Total Revenue, Average Order Value, Total Orders)
Region-wise and category-wise sales analysis
Monthly sales trend and seasonality analysis
Identification of top-selling products
Tactical business improvement suggestions

TOOLS USED:

Python – programming and analysis
Pandas – data manipulation
Matplotlib & Seaborn – visualizations
Google Colab / Jupyter Notebook – interactive environment

DELIVERABLES:

Colab notebook with code and charts
One-page executive summary with insights and recommendations
